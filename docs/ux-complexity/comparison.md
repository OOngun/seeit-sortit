# Before vs After: London Civic Agent

## BEFORE (today)

```
User spots pothole
    ↓
Open browser, find council website           [1 click]
    ↓
Search for "report a problem"                [no link on homepage]
    ↓
Navigate: Services > Roads > Report          [3 clicks]
    ↓
Create account / log in                      [5+ fields]
    ↓
Place pin on map                             [1 click, precision needed]
    ↓
Pick category (67-146 options)               [2 clicks, 2 levels deep]
    ↓
Check for duplicates                         [1 click]
    ↓
Read council criteria                        [1 click]
    ↓
Upload photo                                 [1 click]
    ↓
Write summary + description                  [2 fields]
    ↓
Enter name, email, phone                     [3 fields]
    ↓
Submit                                       [1 click]
    ↓
Wait. Check manually. No follow-up.
Nothing happens.

Total: 8-10 clicks, 7+ form fields, 6 decisions,
       up to 146 categories to navigate,
       and you need to know the right department.
```

**Time: 5-15 minutes.**
**Follow-up: none. You never hear back.**

---

## AFTER (London Civic Agent)

```
User spots pothole
    ↓
Take photo                                   [1 tap]
    ↓
"There's a pothole here"                     [voice note]
    ↓
Done.

Agent handles:
  → Classifies issue from photo
  → Extracts GPS from EXIF
  → Scores severity against London data
  → Identifies correct council + department
  → Files the report
  → Tracks SLA
  → Escalates if ignored
```

**Time: 15 seconds.**
**Follow-up: automatic. Agent tracks, calls, and escalates.**
