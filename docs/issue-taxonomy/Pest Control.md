# Pest Control

**Photo classifiable:** Sometimes (rats, nests)
**Time sensitive:** Moderate (wasp nests = urgent)
**Responsible authority:** Borough environmental health

## Issue Types
- Rat / mouse infestations
- Cockroaches / bedbugs
- Wasp / bee nests
- Pigeon / seagull nuisance
- Fox problems

## Severity Ranking Signals
- Near food premises → higher priority
- Wasp nest near school/playground → urgent
- Bee nests → may need beekeeper, not pest control
- Scale of infestation

## Routes To
- Borough pest control service
- Environmental health (if linked to premises)
- Beekeeper network (bees — protected species)
