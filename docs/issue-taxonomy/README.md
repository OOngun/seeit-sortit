# Issue Taxonomy

Comprehensive taxonomy of civic issues reportable to London borough councils.
Open this folder in Obsidian and enable the Graph View to see the knowledge map.

## Sources
- mySociety Research Report (1.1M+ reports, 2007–2017): research.mysociety.org/html/fms-report/
- mySociety Impact Reports (2023, 2025): research.mysociety.org/html/impact-report-2025/
- FixMyStreet public dashboards: fixmystreet.com/reports
- mySociety open datasets: data.mysociety.org/categories/fixmystreet/
- Camden Council: camden.gov.uk/report-a-problem
- Westminster Council: westminster.gov.uk/report-problem
- Hackney Council: hackney.gov.uk/report
- GOV.UK: gov.uk/report-problems-council

## Volume Distribution (UK-wide, FixMyStreet data)
- Road defects (potholes, paving, markings): ~50% of all reports
- Environmental (litter, fly-tipping, dog fouling): ~25%
- All other categories: ~25%
- Total: ~12,000+ reports/month UK-wide

## Weekly Volumes (top 3)
- Fly-tipping: ~1,550/week
- Graffiti: ~1,020/week
- Potholes: ~870/week
