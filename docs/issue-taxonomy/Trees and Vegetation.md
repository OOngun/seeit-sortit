# Trees and Vegetation

**Photo classifiable:** Yes
**Time sensitive:** Yes (fallen trees, dangerous branches)
**Responsible authority:** Borough parks/trees team

## Issue Types
- Fallen tree blocking road or path
- Dangerous hanging branch
- Overhanging branches onto property
- Dead or diseased tree
- Invasive species (Japanese knotweed, giant hogweed)
- Tree pruning needed
- Damaged grass verges

## Severity Ranking Signals
- Blocking road or pavement → urgent
- Giant hogweed (toxic to skin) → urgent
- Japanese knotweed (legal obligation to control)
- Near power lines or buildings
- Tree preservation order (TPO) status

## Routes To
- Borough trees/parks team (council-owned trees)
- Private landowner (trees on private land — council can serve notice)
- Network Rail (trees near rail lines)
- UK Power Networks (trees near power lines)
